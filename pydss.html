PyDSS

Python wrapper around OpenDSSDirect

2019, https://github.com/NREL/pydss

Source: National Renewable Energy Laboratory (NREL)

Read: Approx. 5 minutes

PyDSS is an opensource software framework developed using Python at National Renewable Energy Laboratory (NREL) in 2019. This tool is essentially a wrapper around OpenDSSDirect which makes it easy to manage and run time series powerflow using OpenDSS.

Organization of input files

PyDSS allows users to organize input files into a folder structure. A PyDSS project skeleton consists of multiple folders as well as subfolders. All the opendss files are stored in `DSSfiles` folder. The exported results will be stored in `Exports` folder. All the log messages will be stored inside the `Logs` folder. Multiple scenarios can be defined within `Scenarios` folder and each folder inside `Scenarios` folder correspond to a single scenario. A scenario folder again consists of multiple folders for controllers, plots, post processing and export settings.

Cosimulation integration

PyDSS uses HELICS for cosimulation. Each PyDSS instance can be cosimulated with other PyDSS instances as well as with instances of other simulators. Multiple simulators of different type can communicate with each other to perform integrated simulation using cosimulation engine such as HELICS. PyDSS makes it easy to cosimulate distribution system with other domains such as transmission system.

Custom control algorithms

PyDSS has built in capabilty to model advanced inverter functions such as voltvar, voltwatt, inverter ride through (2018 as well as 2003) as well as distributed motors. User can also define custom controllers easily. Controllers setting can be defined in a file which PyDSS will load and use during simulation.

Monte carlo, post processing and visualization

Other features of PyDSS include post processing, results visualizationa and monte carlo study. User can customize what plots they want to visualize, what results to export as well as perform sweeping study using MonteCarlo simulation.
